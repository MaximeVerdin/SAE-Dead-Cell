# SAE-Dead-Cell

Ce dépôt contient le code source d’un projet web universitaire (SAE) dédié au jeu rogue-lite populaire *Dead Cells*. Le site sert de vitrine promotionnelle et informative, construit uniquement en HTML et CSS.

## Démo en ligne

Le site est accessible ici :  
https://projets.jrcan.dev/sae-dead-cell/

## Fonctionnalités

Ce site statique est composé de quatre pages principales :

* **Accueil** : Page d’entrée présentant le jeu à travers des sections visuellement attractives. Elle met en avant des aspects clés comme l’exploration, la personnalisation de build et les combats de boss. Un lien vers la page Steam permet de télécharger le jeu.

* **Présentation** : Une page de type article qui détaille l’identité du jeu. Elle couvre son genre (rogue-lite, metroidvania), son développement par Motion Twin, ses mécaniques centrales, sa direction artistique et son succès critique.

* **Boutique** : Une page boutique regroupant des liens vers des plateformes externes où acheter le jeu, ses DLC et divers produits dérivés. Elle inclut :
  * *Jeux et DLC* : Liens vers les pages Steam du jeu de base et de ses quatre extensions majeures (*Fatal Falls*, *The Bad Seed*, *The Queen and the Sea*, *Return to Castlevania*).
  * *Goodies* : Liens vers des produits dérivés, comme des figurines imprimables en 3D et le jeu de société officiel.

* **Mentions légales** : Une page standard regroupant les informations légales du site.

## Pile technologique

* **HTML5** : Structure et contenu des pages.
* **CSS3** : Styles, mise en page, animations et responsive design. Le site utilise des fonctionnalités modernes comme les variables CSS, flexbox et les media queries.
* **Polices personnalisées** : Utilisation de la famille de polices « Berylium » pour une esthétique distinctive.
